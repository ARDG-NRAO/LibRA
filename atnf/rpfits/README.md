This directory contains a subset of the files that make up the RPFITS
source code distribution. They have been reorganized for build convenience,
but details about this package and the original source distribution can be
found at:

http://www.atnf.csiro.au/computing/software/rpfits.html

The files in this directory are covered by the "CSIRO Open Source Software
Agreement (GPLv3+)". You can find the details in the COPYING file found
in the same directory as this file.
